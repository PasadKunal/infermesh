# InferMesh

A production-grade LLM inference gateway and observability platform.

Routes requests across OpenAI, Anthropic, and Gemini with semantic caching,
cost tracking, provider fallback, and a real-time React dashboard.

## Stack
- **Backend**: FastAPI, PostgreSQL, pgvector, Redis, Celery
- **Frontend**: React, Vite, Recharts
- **Infra**: Docker, Railway, GitHub Actions
